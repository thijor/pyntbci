Codes

make_apa_sequence()

Generate an almost perfect auto-correlation sequence.

make_de_bruijn_sequence([k, n, seed])

Generate a de Bruijn sequence.

make_golay_sequence()

Generate complementary Golay sequences.

make_gold_codes([poly1, poly2, seed1, seed2])

Generate a set of Gold codes.

make_m_sequence([poly, base, seed])

Generate a maximum length sequence.

is_de_bruijn_sequence(code[, k, n])

Check whether a code is a de Bruijn sequence.

is_gold_codes(codes)

Check whether the codes are Gold codes.

is_m_sequence(code)

Check whether a code is an m-sequence.

modulate(codes)

Modulate a set of codes.

trials_to_epochs(X, y, codes, epoch_size, ...)

itr(n, p, t)

Compute the information-transfer rate.
