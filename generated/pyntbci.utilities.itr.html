pyntbci.utilities.itr

pyntbci.utilities.itr(n: int | ndarray, p: float | ndarray, t: float | ndarray) → ndarray[source]

Compute the information-transfer rate (ITR).

Parameters:

  • n (int | np.ndarray:) – The number of classes.

  • p (float | np.ndarray:) – The decoding accuracy between 0 and 1.

  • t (float | np.ndarray:) – The decoding time in seconds (including inter-trial time).

Returns:

itr – The ITR in bits per minute.

Return type:

float | np.ndarray
