Compute the covariance matrix.

Parameters:

  • X (NDArray) ‚Äì Data matrix of shape (n_samples, n_features).

  • n_old (int (default: 0)) ‚Äì Number of already observed samples.

  • avg_old (NDArray (default: None)) ‚Äì Already observed average of shape (n_features).

  • cov_old (NDArray (default: None)) ‚Äì Already observed covariance of shape (n_features, n_features).

  • estimator (BaseEstimator (default: None)) ‚Äì An object that estimates a covariance matrix using a fit method. If None, a custom implementation of the empirical covariance is used.

  • running (bool (default: False)) ‚Äì Whether to use a running covariance. If False, the covariance matrix is computed instantaneously using only X, such that n_old, avg_old, and cov_old are not used.

Returns:

  • n_new (int) ‚Äì Number of samples.

  • avg_new (NDArray) ‚Äì The average of shape (1, n_features).

  • cov_new (NDArray) ‚Äì The covariance of shape (n_features, n_features).
